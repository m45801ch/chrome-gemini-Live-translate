# 蜂鳥影片即時翻譯 (Hummingbird Video Live Translate)

蜂鳥影片即時翻譯是一款專為 Chrome 瀏覽器設計的即時影音翻譯擴充套件。本套件能擷取分頁中的影片音訊，透過 Google 最新的 **Gemini Live API** 進行低延遲的語音辨識與翻譯，並直接在影片播放器上渲染出美觀、支援雙語/單語且可自由調整高度與字型樣式的懸浮字幕。

## 🌟 核心功能 (v1.1.3)

*   **即時語音翻譯**：一鍵擷取影片音訊，透過 WebSocket 與 Gemini Live API 連線進行流式（Streaming）即時雙語翻譯。
*   **精美簡約介面與質感排版**：全新升級的白色系質感設計，優化彈出視窗（Popup UI）順序，將「啟動影片即時語音翻譯」移至最上方以便一鍵啟動。
*   **獨立選項設定頁面 (Options Page)**：
    1.  🔑 **Gemini API 金鑰設定**：提供 API Key 輸入與快捷前往 Google AI Studio 申請的引導按鈕。
    2.  🌐 **翻譯語言與快取清除設定**：提供語音來源語言與翻譯目標語言設定，並支援「清理快取對話間隔」選項，定時自動重啟連線清空語音模型快取，有效防止延遲。
    3.  🎨 **字幕樣式自訂**：可即時調整主/譯文字幕大小、字型、粗細、顏色、背景透明度與最大行數。
*   **YouTube 播放器深度整合**：
    1.  **全新控制欄圖標設計**：採用滿版炭灰色圓角按鈕底色，設計三個發光霓虹線條蜂鳥圖標（即時翻譯 ON 為翠綠色、即時翻譯 OFF 為鮮紅色、內建字幕 ON 為蔚藍色 ＋ CC 標籤），完美融入播放器控制欄。
    2.  設定選單與下載子頁面全面引進簡約的描邊圖標（Icon），提升視覺質感。
    3.  點擊選單外部任意空白處自動收合選單。
*   **字幕渲染與顯示優化**：
    1.  **歷史字幕輕量化**：已播放的歷史字幕（上一句或上兩句）字型大小自動縮小為 90%，字型粗細減少一級（若原為 400 則降為 300，最少為 300），顏色改為淡灰色，有效區分當前講話內容。
    2.  **無縫融入背景**：當背景透明度拉到 0%（即完全透明）時，會自動移除邊框與陰影，讓字幕無感融入影片中。
    3.  **防止文字溢出與殘留**：直接對字幕資料進行切片（`.slice(-maxLines)`），保證 DOM 內只渲染最大行數，徹底解決上方殘留被截斷的微小字母碎屑問題。
*   **控制選單整合「字幕下載」**：
    1.  將原設定面板內冗餘的下載按鈕移除，統一整合至播放器選單主頁面的「字幕下載」入口。
    2.  提供三種下載選項：**「下載原文字幕 (.srt)」**、**「下載譯文字幕 (.srt)」**與**「下載雙語對照字幕 (.srt)」**，並支援線上下載前自動批次翻譯未翻譯行數。
*   **無音訊自動隱藏與清除**：當影片無音訊達 7 秒時，自動關閉並隱藏字幕框，同時徹底清空快取中的歷史字幕緩衝區，確保下次偵測到語音時不會殘留舊有的文字。
*   **字幕拖曳定位與滑鼠游標鎖定**：支援在影片播放時用滑鼠隨意拖曳字幕至合適的高度，採相對位移計算防止跳動，並在拖曳時引入全螢幕遮罩鎖定抓取游標。

## 🛠️ 開發與建置說明

本專案使用 **WXT (Next-gen Web Extension Framework)** 搭配 **React** 與 **TypeScript** 進行開發。

### 安裝依賴項目
```bash
npm install
```

### 啟動開發伺服器 (Dev Mode)
開發伺服器會自動載入並監聽變更：
```bash
npm run dev
```

### 編譯生產版本 (Build for Production)
將專案打包編譯為適用於 Chrome 的 MV3 擴充套件包：
```bash
npm run build
```
編譯後的檔案將存放在 `.output/chrome-mv3/` 目錄下。

---

## 📦 安裝步驟 (手動載入未封裝擴充套件)

1.  在專案根目錄下執行生產建置命令：
    ```bash
    npm run build
    ```
2.  開啟 Chrome 瀏覽器，進入擴充功能管理頁面：`chrome://extensions/`
3.  開啟右上角的 **「開發人員模式」**。
4.  點擊左上角的 **「載入未封裝項目」**。
5.  選擇專案目錄下的 `.output/chrome-mv3` 資料夾。
6.  安裝完成後，即可在右上角看到「蜂鳥影片即時翻譯」圖示。

---

## 🚀 使用指南

1.  點擊瀏覽器右上角的**蜂鳥圖示**開啟 Popup 視窗。
2.  點擊右上角的**齒輪圖示**開啟設定分頁，在 **Gemini API 金鑰** 欄位中輸入您的 Gemini API Key 並存檔。
3.  開啟 YouTube 或任何影片網站，點擊播放器右下角的蜂鳥按鈕。
4.  點擊 **「開啟即時翻譯」**，當 ON 綠色狀態燈亮起後即可即時接收並顯示翻譯字幕。
5.  您隨時可以點擊播放器按鈕中的 **「字幕樣式」** 或右上角設定頁面的 **「字幕設定」** 來精細調節字體與顏色，設計出最舒適的閱讀體驗！

## 🔒 隱私權政策 (Privacy Policy)

「蜂鳥影片即時翻譯」為開源專案，我們極力保護使用者的資料隱私。本套件遵循以下資料處理原則：

*   **API 金鑰安全**：您在設定頁面中輸入的所有 API 金鑰（例如 Gemini API Key）均安全地儲存在您當地的瀏覽器本機儲存空間（`chrome.storage.local`）中，本套件不會將金鑰傳輸至任何第三方或本專案開發團隊。
*   **音訊串流處理**：啟動「即時語音翻譯」時，本套件僅擷取該瀏覽器分頁的音軌，並透過加密的 WebSocket 連線直接傳送至 Google 官方 Gemini API 端點（`generativelanguage.googleapis.com`）以進行即時的語音辨識與翻譯。
*   **零資料收集**：我們不會在本地瀏覽器之外的任何伺服器儲存、收集或傳輸您的任何隱私資料（包括但不限於音訊內容、翻譯文本、瀏覽記錄或個人識別資訊）。所有翻譯流程均在您的本機與官方 API 之間點對點完成。
