# 蜂鳥影片即時翻譯 (Hummingbird Video Live Translate)

蜂鳥影片即時翻譯是一款專為 Chrome 瀏覽器設計的即時影音翻譯擴充套件。本套件能擷取分頁中的影片音訊，透過 Google 最新的 **Gemini Live API** 進行低延遲的語音辨識與翻譯，並直接在影片播放器上渲染出美觀、支援雙語/單語且可自由调整高度與字型樣式的懸浮字幕。

## 🌟 核心功能

*   **即時語音翻譯**：一鍵擷取影片音訊，透過 WebSocket 與 Gemini Live API 連線進行流式（Streaming）即時雙語翻譯。
*   **精美簡約白色介面**：全新升級的 Light Mode 白色系質感設計，包含右上角 Popup 視窗、YouTube 播放器控制選單及獨立設定分頁。
*   **獨立選項設定頁面 (Options Page)**：
    1.  🔑 **Gemini API 金鑰設定**：提供 API Key 輸入與快捷前往 Google AI Studio 申請的引導按鈕。
    2.  🌐 **翻譯語言設定**：提供語音來源語言（支援自動偵測）與翻譯目標語言，支援包含繁體中文等 70+ 種主要語言。
    3.  🎨 **字幕樣式自訂**：可即時調整主/譯文字幕的大小、字型、粗細、顏色、背景透明度與最大行數。
*   **YouTube 播放器深度整合**：
    1.  在 YouTube 播放器右下角控制欄自動嵌入去背蜂鳥按鈕與直觀的 ON / OFF 膠囊狀態燈。
    2.  點擊播放器選單內的「字幕樣式」可以直接在影片選單內即時設定樣式。
    3.  支援點擊播放器選單外部的任意空白處自動收合選單。
*   **字幕自由定位**：在影片播放時，可用滑鼠隨意拖曳字幕至合適的高度，避免遮擋影片內容。

## 🛠️ 開發與建置說明

本專案使用 **WXT (Next-gen Web Extension Framework)** 搭配 **React** 與 **TypeScript** 進行開發。

### 安裝依賴項目
```bash
npm install
```

### 啟動開發伺服器 (Dev Mode)
開發伺服器會自動載入並監聽變更：
```bash
npm run dev
```

### 編譯生產版本 (Build for Production)
將專案打包編譯為適用於 Chrome 的 MV3 擴充套件包：
```bash
npm run build
```
編譯後的檔案將存放在 `.output/chrome-mv3/` 目錄下。

---

## 📦 安裝步驟 (手動載入未封裝擴充套件)

1.  在專案根目錄下執行生產建置命令：
    ```bash
    npm run build
    ```
2.  開啟 Chrome 瀏覽器，進入擴充功能管理頁面：`chrome://extensions/`
3.  開啟右上角的 **「開發人員模式」**。
4.  點擊左上角的 **「載入未封裝項目」**。
5.  選擇專案目錄下的 `.output/chrome-mv3` 資料夾。
6.  安裝完成後，即可在右上角看到「蜂鳥影片即時翻譯」圖示。

---

## 🚀 使用指南

1.  點擊瀏覽器右上角的**蜂鳥圖示**開啟 Popup 視窗。
2.  點擊右上角的**齒輪圖示**開啟設定分頁，在 **Gemini API 金鑰** 欄位中輸入您的 Gemini API Key 並存檔。
3.  開啟 YouTube 或任何影片網站，點擊播放器右下角的蜂鳥按鈕。
4.  點擊 **「開啟即時翻譯」**，當 ON 綠色狀態燈亮起後即可即時接收並顯示翻譯字幕。
5.  您隨時可以點擊播放器按鈕中的 **「字幕樣式」** 或右上角設定頁面的 **「字幕設定」** 來精細調節字體與顏色，設計出最舒適的閱讀體驗！
